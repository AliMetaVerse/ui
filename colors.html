Usage Guidelines

Best practices for using Webropol colors in your designs to maintain brand consistency and accessibility.

Do's

  • • Use Webropol Teal (#14b8a6) as the primary call-to-action color
  • • Maintain sufficient contrast ratios for accessibility (4.5:1 minimum)
  • • Use lighter shades for backgrounds and hover states
  • • Apply semantic colors consistently (green for success, red for errors)
  • • Use the gray scale for text hierarchy and neutral elements

Don'ts

  • • Don't use colors outside the approved palette
  • • Don't use multiple primary colors in the same component
  • • Don't apply colors with insufficient contrast
  • • Don't override semantic color meanings
  • • Don't use pure black (#000000) or pure white (#ffffff) for text

Accessibility Guidelines

All color combinations in this palette meet WCAG 2.1 AA standards for contrast accessibility. When creating new combinations, always verify contrast ratios.

Normal text: Minimum 4.5:1 contrast ratio
Large text: Minimum 3:1 contrast ratio

          50: '#eff6ff',

          900: '#1e3a8a',

CSS Custom Properties

:root {
  --webropol-teal-50: #f0fdfa;
  --webropol-teal-500: #14b8a6;
  --webropol-teal-900: #134e4a;
