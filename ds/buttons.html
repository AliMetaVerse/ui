Usage Guidelines

Do's

  • Use primary buttons for main actions
  • Keep button text concise and action-oriented
  • Maintain consistent spacing around buttons

Don'ts

  • Use too many primary buttons on one page
  • Mix different button styles in the same section
  • Use disabled buttons without clear indication why
