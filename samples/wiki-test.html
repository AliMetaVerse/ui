Survey Creation & Access

Steps to create a new survey:

  1. Click the Create New button.
  2. Enter a survey title.
  3. Select a folder to save the survey.
  4. Choose a layout template.
  5. Click Create Survey.

Reporting

The reporting tab allows users to analyze survey responses using various visualization tools such as charts and graphs.
